# PIC2LCEDA
立创EDA_PCB照片生成器，作者：矛盾聚合体  
使用文档请看[PCB照片_附带生成脚本](https://lceda.cn/Knight_Sin/PCBzhao-pian)
# 使用
## 方法一：下载程序
windows、Linux已经编译好的程序见Release
## 方法二：自行编译
### 环境准备
- 安装python3
- 安装依赖包：opencv、numpy
### 克隆代码、运行
```bash
$ git clone githubUrl
$ cd PIC2LCEDA
$ python3 PIC2LCEDA.py
```
